# Super Market Bill Generation System

## Project Overview
Super Market Bill Generation System is a Python-based application that helps generate bills for customers by calculating the total cost of purchased items.

## Features
- Display Product List
- Add Multiple Products
- Calculate Total Amount
- Generate Customer Bill
- Easy Console-Based Interface

## Technologies Used
- Python 3

## How to Run

1. Install Python 3
2. Open supermarket_bill.py in Python IDLE
3. Run the program using F5
4. Enter product names and quantities
5. View the generated bill

## Sample Products

- Rice - ₹60/kg
- Sugar - ₹45/kg
- Milk - ₹30
- Bread - ₹40
- Biscuit - ₹20
